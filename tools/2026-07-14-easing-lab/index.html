      三种引擎,都是真算的。 Bézier 用 Newton–Raphson 解 x(t)=进度(WebKit UnitBezier 同款);Spring 对二阶弹簧微分方程数值积分求整条位移曲线并自动判定收敛时长;Steps 按 CSS steps() 规范逐段取值。弹簧/自定义曲线导出为现代 CSS linear(),回弹也能纯 CSS 还原。全程在你浏览器里运行,曲线保存在 localStorage。
